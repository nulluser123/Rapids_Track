# Project Brief: Rapid Tracker v2 (Grandmaster's Ledger)

## Core Requirements & Scope
Rapid Tracker v2 is a frontend web application designed to track, store, and analyze Chess.com rapid ratings for a curated list of players.

## Goals
- Operate as a Single Page Application (SPA).
- Provide high-quality visual aesthetics (premium dark mode, glassmorphic UI).
- Real-time data fetching from Chess.com public API.
- Deep statistical analysis of player performance.
